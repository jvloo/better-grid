# Internal docs

Contributor reference. None of this is required to use Better Grid.

| Path                                          | Purpose                                                                                |
| --------------------------------------------- | -------------------------------------------------------------------------------------- |
| [`v1-init-api-history.md`](v1-init-api-history.md) | Design history of the v1 init API — what changed across pre-release iterations.    |
| [`pr-summaries/`](pr-summaries/)              | Per-PR detailed summaries for the bigger merges. More detail than `CHANGELOG.md`.      |
| [`specs/`](specs/)                            | Brainstorming specs for major feature work, dated `YYYY-MM-DD-<topic>-design.md`.      |
| [`plans/`](plans/)                            | Implementation plans corresponding to the specs, dated `YYYY-MM-DD-<topic>-redesign.md`. |

Specs and plans are written by the brainstorming + writing-plans skills. New artifacts go under `internal/specs/` and `internal/plans/` respectively.
